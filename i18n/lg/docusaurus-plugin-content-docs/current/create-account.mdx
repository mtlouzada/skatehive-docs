## 2. Londa erinnya lyo
Kitya erinnya lyo eriddamu okuleeta okusinziira ku ngeri gy'olina ekisunsulirwa ekikulu.

![Londa erinnya lyo n’emuuyo](https://hackmd.io/_uploads/H1tZGNP1Jx.png)

## 3. Oluvannyuma lwa kuddiza ku **"Looks Good, Lets go for it!"**, ekizibu ky’okusaba muakaunti kijejja okuva mu kifu ky’oyagala okukakasa. Olw’okunyweza, akaunti y’okujja ne ey’okuva ku kikwatibwa.
